# Digital Downloads (ASP.NET Core + Stripe + Azure Blob)

**Overview**
A sample full-stack web app to sell digital products using Stripe for payments and Azure Blob Storage for storing downloadable assets. Built with ASP.NET Core.

**Phases**
- Phase 1: Project setup (this commit)
- Phase 2: Models and DB (EF Core)
- Phase 3: Stripe integration
- Phase 4: Azure Blob Storage upload/download
- Phase 5: Authentication, admin and customer portals
- Phase 6: Deployment and security hardening

**Getting Started (local)**
1. Clone repo
2. Set your local secrets (Stripe keys, Azure connection string) as environment variables.
3. `dotnet run --project DigitalDownloads.Web`
